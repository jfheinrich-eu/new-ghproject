# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-04

### Added
- Initial release of New-GHProject module
- Core function `New-GHProject` to create GitHub repositories
- Automatic branch protection configuration
- Support for both public and private repositories
- Comprehensive Pester test suite
- PSScriptAnalyzer code quality checks
- GitHub Actions CI/CD workflows
- Dependabot configuration for automated dependency updates
- PowerShell Gallery deployment workflow
- Comprehensive documentation in README

### Features
- Cross-platform support (Windows, Linux, macOS)
- ShouldProcess support for safer operations
- Detailed help documentation
- Error handling and validation
